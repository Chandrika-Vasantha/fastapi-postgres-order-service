# FastAPI Postgres Order Service

Production-style FastAPI backend service for managing customer orders using REST APIs.

## Features

* FastAPI REST APIs
* Order creation and retrieval
* JSON request/response handling
* Clean project structure
* Git version control
* Docker-ready setup
* Scalable backend foundation

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* Docker
* GitHub

## API Endpoints

### Get API Status

```http
GET /
```

### Get Orders

```http
GET /orders
```

### Create Order

```http
POST /orders
```
